# Arcade Game

A simple browser-based Space Invaders game built with HTML, CSS, and JavaScript.

## How to Play
- Use the left and right arrow keys to move.
- Press the space bar to shoot.
- Defeat all the aliens before they reach you.

## Run Locally
Open the game file in a browser, or serve the folder with a simple local server.

## Deploy to GitHub Pages
Upload the project to GitHub and enable GitHub Pages to make it available online.
